# CoffeemaniaGeographyService (C# / ASP.NET Core)

Небольшой API-проект для расчёта расстояния в километрах между двумя географическими точками, заданными широтой и долготой. Проект предназначен как тестовое задание.

---

## Задание

> Необходимо реализовать проект расчета расстояния в километрах между двумя точками, точки передаются как входные параметры в запросе.

---

## Назначение

API принимает две точки на поверхности Земли и возвращает расстояние между ними **в километрах**.

---

## Технологии

- **.NET / C#**
- **ASP.NET Core Web API**
- **nUnit** — юнит-тесты
- **FluentAssertions** — удобные и читаемые ассерты
- **Haversine formula** — алгоритм расчёта расстояния

---

## Используемый алгоритм

Для расчёта расстояния используется **формула гаверсинусов (Haversine formula)**, которая позволяет определить длину кратчайшей дуги между двумя точками на сфере.

Формула учитывает кривизну Земли и даёт достаточно точный результат для большинства практических задач.

Радиус Земли принят равным: 6371 км.
